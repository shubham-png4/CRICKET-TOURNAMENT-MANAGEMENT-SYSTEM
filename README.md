# Cricket Tournament Management & Analytics System (SQL)

A structural relational database system built to store, manage, and analyze local cricket tournament metrics. This project maps real-world sports statistics into optimized database entities and showcases advanced multi-table joins, aggregations, and data calculations.

## 📊 Database Relationship Architecture

The data structure relies on a 1-to-Many relational workflow:
- **Teams** can have multiple **Players**.
- **Players** can possess multiple individual **MatchPerformances**.

## 🛠️ Tech Stack & Requirements
- **Language:** SQL
- **RDBMS Engine Support:** MySQL / PostgreSQL / MS SQL Server
